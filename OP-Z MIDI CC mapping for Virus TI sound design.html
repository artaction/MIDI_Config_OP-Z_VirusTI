The purpose of this is to create a set of controls, that when given random values, provide some interesting changes in timbre to the voices on the Virus TI. When using the OP-Z for generating random values, we are limited to 16 MIDI cc parameters per track. I have copied the MIDI CC controls that I think might be best suited for new voice / patch creation on the Virus TI.
